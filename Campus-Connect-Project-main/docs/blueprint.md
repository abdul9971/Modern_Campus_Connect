# **App Name**: CampusConnect

## Core Features:

- Role Selection: Allows users to select their role (Admin, Teacher, Student) upon launching the application.
- Admin Dashboard: Provides Admin users with a central control panel for managing teachers, students, and timetable.
- Teacher Management: Allows Admin to add new teachers by entering personal details and creating login credentials.
- Student Management: Allows Admin to add new students by entering personal details and assigning them to classes.
- AI-Powered Timetable Extraction: Uses the Gemini API as a tool to extract timetable data from images or manual input, enabling Admin to edit and save data into Firestore.
- Attendance Marking: Allows Teachers to mark student attendance (Present, Absent, Reset) for their assigned lectures.
- Real-Time Attendance Tracking: Provides Students with real-time updates on their attendance status based on Teacher actions.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), chosen to evoke a sense of trust and knowledge.
- Background color: Light gray (#F0F0F0), providing a neutral and clean backdrop.
- Accent color: Soft violet (#EE82EE), offering a contrasting highlight for interactive elements.
- Body and headline font: 'Inter', a sans-serif font with a modern, machined look appropriate for educational information.
- Use clean and consistent icons representing roles, actions, and data visualization.
- Web-first, mobile-friendly layout designed for intuitive navigation and access to core features.
- Subtle transition effects to enhance user experience during attendance updates and navigation.
